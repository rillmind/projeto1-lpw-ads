# Projeto de Programação para Internet

## Rodar

Para rodar o projeto basta clonar o repositório:

```sh
git clone https://github.com/rillmind/projeto1-lpw-ads
cd projeto1-lpw-ads
```

Baixar as dependencias:

```sh
bun install
```

ou com node:

```sh
npm install
```

E, para de fato rodar:

```sh
bun .db/setup.js
bun start
```
